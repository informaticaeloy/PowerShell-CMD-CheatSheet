## DESCRIPCION

Colección de scripts y utilidades en PowerShell y CMD orientadas a la automatización, administración e integración con Zabbix. Incluye herramientas para importación masiva, generación de configuraciones, gestión de hosts y tareas operativas relacionadas con el entorno de monitorización.
